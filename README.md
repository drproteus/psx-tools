# PSX Tools

Some tools and scripts for working with PSX roms.

## Scripts
* [m3u_gen](/blob/master/scripts/m3u_gen.py): generate m3u files for all disc sets in folder

## Tools
* `./main.py detect-title $FILENAME`: get the Title ID from the disc file
* `./main.py lookup-disc $FILENAME`: lookup the game information by its Title ID from [SerialStation](https://serialstation.com/)
